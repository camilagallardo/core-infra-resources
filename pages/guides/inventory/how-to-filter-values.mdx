# Function getFilterValues

In order to get `filter values` for a given product we need to use Inventory service(INV). Our API allows easy access by a `GET` request to our `filter-values` endpoint.


### HTTP VERB: `GET`
### URL: https://api.psrestful.com/v2.0.0/suppliers/{SUPPLIER_CODE}/filter-values/{PRODUCT_ID}/

### Example Response:

This is the response from calling with:   
`SUPPLIER_CODE` = `HIT`   
`PRODUCT_ID` = `55414`

**COMPLETE URL**: 
```url copy
https://api.psrestful.com/v2.0.0/suppliers/HIT/filter-values/55414/
```
