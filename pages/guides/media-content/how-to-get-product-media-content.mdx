# Function getMediaContent

In order to get `all media content` for a given product we need to use Media Content Service(MED). Our API allows easy access by a `GET` request to our `medias` endpoint.


### HTTP VERB: `GET`
### URL: https://api.psrestful.com/v2.0.0/suppliers/{SUPPLIER_CODE}/medias/{PRODUCT_ID}/

### Example Response:

This is the response from calling with:   
`SUPPLIER_CODE` = `HIT`   
`PRODUCT_ID` = `5989`

**COMPLETE URL**: 
```url copy
https://api.psrestful.com/v1.1.0/suppliers/HIT/medias/5989/
```
